## 🏮 프로젝트 소개

**행복한 한가위 - 연등 잡기 게임**은  
Vue 3 + TypeScript + Vite 기반으로 제작된 한가위(추석) 테마의 미니 게임입니다.  
하늘에서 내려오는 연등을 클릭해 잡으며 점수를 얻고,  
연속으로 잡을수록 콤보 보너스가 올라갑니다! 🎇

귀여운 달토끼와 은은한 달빛, 그리고 반짝이는 연등들이  
가을 밤의 따뜻한 명절 분위기를 전해줍니다.

---

## ✨ 주요 기능

| 기능 | 설명 |
|------|------|
| 🎮 **연등 잡기 인터랙션** | 마우스로 연등을 클릭하면 반짝이는 효과와 함께 점수가 올라갑니다. |
| ⏱️ **타이머 & 점수 시스템** | 제한 시간 60초 동안 가능한 한 많은 연등을 잡으세요! |
| 🔥 **콤보 보너스** | 연속 성공 시 추가 점수! 실수하면 콤보가 초기화됩니다. |
| 🌕 **달빛 애니메이션** | 은은하게 빛나는 달과 하늘 배경으로 몰입감 UP |
| 🧨 **반짝이는 파티클 효과** | 연등이 터질 때마다 빛이 흩날리며 명절의 화려함을 표현합니다. |
| 🎨 **한가위 테마 디자인** | 전통 색감의 연등, 달빛, 잔디밭, 밤하늘을 따뜻한 색조로 구현했습니다. |
